# ZAINEX

Fresh Laravel API + Next.js React frontend monorepo.

## Structure

- backend: Laravel JSON API
- frontend: Next.js, React, TypeScript and Tailwind
- RUN_ZAINEX_DEV.ps1: Starts both development servers
- BUILD_ZAINEX.ps1: Runs Laravel tests, lint, TypeScript and production build

## Local addresses

- Frontend: http://localhost:3000
- Laravel API: http://127.0.0.1:8000
- Health API: http://127.0.0.1:8000/api/health
- Markets API: http://127.0.0.1:8000/api/markets
